# Pythia

Pythia is an application that executes code safely, within UML virtual machines.

## Requirements

- Make 4.0 or later
- Go 1.2.1 or later

## Quick Install

- Clone the repository

        $ git clone --recursive https://github.com/pythia-project/pythia.git

- Get into the directory

        $ cd pythia

- Launch installation

        $ make
